My Resume Template

Dokumentasi Tema Blogger "My Resume"

Versi Terbaru: v1.2

Instalasi

  1. Download file tema dalam format .zip dari link resmi.
  2. Ekstrak file .zip di komputer Anda.
  3. Di dalam folder hasil ekstrak, buka file utama myresume.xml.
  4. Buka Blogger ‚Üí Tema ‚Üí klik panah bawah pada tombol "Sesuaikan" ‚Üí pilih Edit HTML.
  5. Hapus semua kode lama (Ctrl + A ‚Üí Delete).
  6. Salin semua isi file .xml ‚Üí tempelkan ke editor HTML Blogger.
  7. Klik Simpan Tema.
  8. Refresh blog Anda untuk melihat hasilnya.

Catatan: Sebaiknya lakukan instalasi di blog kosong atau buat blog baru agar sesuai dengan demo template

Persyaratan

  • Platform: Blogger terbaru (2025)
  • Browser: Chrome, Edge, Firefox, Safari versi terbaru
  • Resolusi minimum: 320px (mobile)

Cara Update ke Versi Terbaru

  1. Backup tema lama di Blogger ‚Üí Tema ‚Üí Backup.
  2. Download versi terbaru dari halaman Release.
  3. Buka Edit HTML ‚Üí ganti kode dengan versi baru.
  4. Sesuaikan kembali widget (nama, foto, link sosmed).

Struktur Layout

  • Kiri: Informasi profil, foto, nama.
  • Kanan: Kontak, keahlian, resume download, deskripsi singkat.
  • Section tambahan: Self-Learning, Keahlian, Portfolio, Footer.

Kustomisasi Warna & Tampilan

  • bodyColor: Warna body
  • darkColor: Warna teks gelap
  • lightColor: Warna teks terang
  • accentColor: Warna aksen (judul, link, button)
  • bg1Color - bg4Color: Background section berbeda
  • bgDarkColor - bgLightColor: Background section berbeda
  • borderColor: Border foto profil dan elemen aksen

Widgets & Bagian yang Bisa Diedit

  • Nama dan Foto: Widget HTML1
  • Kontak: Widget HTML2
  • Download Resume: Widget HTML3
  • Profil Singkat: Widget HTML4
  • Self-Learning: Widget HTML5
  • Keahlian dan Kemampuan: Widget HTML6
  • Portfolio: Widget HTML7-10

Menambahkan & Mengganti Ikon Sosial Media

Template ini menggunakan Font Awesome untuk ikon. Contoh:

<a href="https://twitter.com/username" target="_blank">
  <i class="fab fa-x-twitter"></i> Twitter
</a>

Ganti fab fa-x-twitter dengan ikon lain dari Font Awesome Icons.

Mengatur Progress Bar Skill

Contoh menambahkan skill:

<div class="progress-item">
  <span>HTML</span>
  <div class="progress-bar">
    <div class="progress-fill" style="width:90%"></div>
  </div>
</div>

Ubah width:90% sesuai kemampuan yang ingin ditampilkan.

Menambahkan Portfolio

<a class="portfolio-item" href="LINK_PROJECT">
  <img src="GAMBAR_PROJECT" alt="Nama Project">
  <h3>Judul Project</h3>
  <p>Deskripsi Project</p>
</a>
