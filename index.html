My Resume Template

Dokumentasi Tema Blogger "My Resume"

Instalasi

  1. Download file tema dalam format .zip dari link resmi.
  2. Ekstrak file .zip di komputer Anda.
  3. Di dalam folder hasil ekstrak, buka file utama myresume.xml dengan text editor seperti Notepad atau VS Code.
  4. Buka Blogger → Tema → klik panah bawah pada tombol "Sesuaikan" → pilih Edit HTML.
  5. Hapus semua kode lama (Ctrl + A → Delete).
  6. Salin semua isi file .xml → tempelkan ke editor HTML Blogger.
  7. Klik Simpan Tema.
  8. Refresh blog Anda untuk melihat hasilnya.

Catatan: Sebaiknya lakukan instalasi di blog kosong atau buat blog baru agar sesuai dengan demo template

Struktur Layout

  • Kiri: Informasi profil, foto, nama.
  • Kanan: Kontak, keahlian, resume download, deskripsi singkat.
  • Section tambahan: Self-Learning, Keahlian, Portfolio, Footer.

Kustomisasi Warna & Tampilan

  • bodyColor: Warna body
  • darkColor: Warna teks gelap
  • lightColor: Warna teks terang
  • accentColor: Warna aksen (judul, link, button)
  • bg1Color - bg4Color: Background section berbeda
  • borderColor: Border foto profil dan elemen aksen

Widgets & Bagian yang Bisa Diedit
